# About and Thesis

This directory contains the foundational materials, architectural thesis, and core reflections for the Panaversity Agent Factory curriculum. The content captured here establishes the vocabulary, principles, and invariants that govern the design, manufacturing, deployment, and operation of Digital Full-Time Equivalents (Digital FTEs) within an AI-Native Company.

## Directory Contents

The materials in this directory are structured as follows:

*   **about_notes.md**: Detailed notes derived from the Agent Factory "About" documentation. This file describes the four-channel ecosystem of the Agent Factory (The Book, TutorClaw, the Skillpack, and specialized Derivative Books) and outlines the roles of various team members (Builders, Knowledge Holders, Decision Makers, Translators, and Operators) in constructing the Agentic Enterprise.
*   **thesis_notes.md**: A comprehensive transcription and breakdown of the Agent Factory Thesis. This file outlines the transition from software subscriptions to outcome-based results, defines the Two-Layer Model, contrasts the Two Modes of General Agent Use (Problem-Solving and Manufacturing), explains the 10-80-10 operating rhythm, and documents the Seven Invariants of the Agent Factory alongside the 2026 Reference Stack.
*   **reflections.md**: Philosophical reflections and critical analyses concerning the socio-economic impact of the Agent Era, the shifting boundaries of professional identity for engineers and domain experts, and the implications of agents acting as autonomous economic actors.

## Core Concepts

The documentation in this folder centers around several core tenets:

1.  **Digital FTE (AI Worker / AI Employee)**: Role-based AI agents configured to perform structured knowledge work continuously inside organizational environments under human supervision.
2.  **The Two-Layer Model**: A structure comprising the Edge Layer (composed of sovereign personal agents, or Identic AI, representing individual human judgment) and the AI Workforce Layer (composed of specialized AI Workers executing organizational workflows).
3.  **The 10-80-10 Operating Rhythm**: An execution pattern where humans spend 10% of their time setting intent, AI agents perform execution for 80% of the duration, and humans return for the final 10% to verify and refine the outcome.
4.  **The Seven Invariants**: Architectural rules that must remain true across any realization of an Agent Factory, ensuring the system remains governable, durable, and coherent.

For detailed concepts and technical implementations, refer to the respective notes files in this directory.
