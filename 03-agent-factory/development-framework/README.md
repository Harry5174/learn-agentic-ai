# Agent Factory Development Framework (AFDF) — Historical Snapshot

> **Status:** This directory is a frozen historical AFDF snapshot.
>
> For current framework use, go to the [Agent Factory Engineering Framework repository](https://github.com/Harry5174/agent-factory-engineering-framework).

The in-repository Agent Factory Development Framework (AFDF) was the valid framework location during earlier Agent Factory work. AFEF is its canonical successor, and AFEF v0.1.0 established the first canonical public baseline.

Consult the [AFEF Releases page](https://github.com/Harry5174/agent-factory-engineering-framework/releases) to determine the current released version. The v0.1.0 baseline is preserved at the [AFEF v0.1.0 release](https://github.com/Harry5174/agent-factory-engineering-framework/releases/tag/v0.1.0).

## Historical Contents

This directory remains intact for provenance and historical study. It contains:

- Original AFDF specifications, protocols, and templates.
- Historical framework evidence and status snapshots.
- Historical Agent Factory project memory.
- Artifact bootstrap examples.
- Source material from which AFEF was selectively extracted.

Do not start a new framework session from this directory or its templates. See [LEGACY.md](LEGACY.md) for the complete transition record, freeze rules, canonical links, and project-memory boundary.
